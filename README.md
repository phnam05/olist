## Overview
This project is an interactive **Streamlit dashboard** for analyzing the Brazilian **Olist e-commerce dataset (2016–2018)**.

It explores:
- 📈 Order trends over time
- 💳 Payment behavior and installment usage
- 🚚 Delivery performance and customer satisfaction
- 🔁 Customer retention patterns

---

## ✨ Features

### 🔢 Key Metrics
- Total Orders
- Total Revenue
- Average Review Score
- Late Delivery Rate
- Customer Retention Rate

### 📈 EDA
- Monthly trends with time slider
- Year-over-year comparison
- Order status breakdown

### 💳 Payment Behavior
- Payment distribution
- Avg order value
- Installment usage

### 🚚 Delivery & Satisfaction
- Late vs on-time comparison
- Review score impact
- Lateness severity buckets

### 🔁 Customer Retention
- One-time vs returning users
- Time-to-second purchase
- Retention distribution
